# Roadmap

## Q1 Goals
- Finalize core API endpoints for translation, forecasting, decoding, math, and design assistance.
- Publish baseline benchmarks comparing OmniRosetta modules to open-source baselines.

## Q2 Goals
- Introduce reinforcement learning from domain experts for improved interpretability.
- Expand dataset coverage in the `data/` directory with curated corpora for underrepresented scripts.

## Q3 Goals
- Release community toolkit for fine-tuning custom engines using Omni AI Kernel hooks.
- Launch public leaderboard for shared forecasting scenarios.

Contributions and feedback on the roadmap are welcome via GitHub issues or community forums.
